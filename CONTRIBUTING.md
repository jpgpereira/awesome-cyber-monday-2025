## ğŸ”„ Pull Request Process

1. **Fork the repository**
   ```bash
   git clone https://github.com/punkpeye/awesome-cyber-monday-2025.git
   ```

2. **Create a new branch**
   ```bash
   git checkout -b add-deal-productname
   ```

3. **Make your changes**
   - Add your deal to the appropriate category
   - Maintain alphabetical order within categories
   - Follow the existing formatting

4. **Commit your changes**
   ```bash
   git add README.md
   git commit -m "Add [Product Name] deal to [Category]"
   ```

5. **Push to your fork**
   ```bash
   git push origin add-deal-productname
   ```

6. **Submit a Pull Request**
   - Use a clear, descriptive title
   - Reference any related issues
   - Provide context for your submission
